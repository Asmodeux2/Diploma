# Diploma Skill for AI

Специализированный Skill для подготовки выпускных квалификационных работ (ВКР), дипломных проектов и научных работ в соответствии с требованиями российских вузов и действующих стандартов оформления.

## Возможности

Данный Skill позволяет использовать современные ИИ-модели для:

* написания дипломных работ;
* подготовки разделов ВКР;
* формирования введения и заключения;
* разработки структуры работы;
* подготовки обзора литературы;
* написания аналитических разделов;
* описания проектирования и реализации программных систем;
* подготовки материалов по ГОСТ;
* проверки научного стиля текста;
* формирования выводов по главам.

## Поддерживаемые стандарты

* ГОСТ 7.0.100-2018
* ГОСТ Р 7.0.5-2008
* требования российских вузов
* методические рекомендации кафедр
* требования МИРЭА (базовая версия)

## Для каких направлений подходит

* Программная инженерия
* Информатика и вычислительная техника
* Прикладная информатика
* Информационные системы и технологии
* Информационная безопасность
* Анализ данных
* Искусственный интеллект
* Автоматизация и управление

## Использование

### ChatGPT

Добавьте содержимое `Skill.md` в системный промпт проекта или пользовательские инструкции.

### Claude

Используйте содержимое файла как Project Instructions.

### Cursor

Добавьте Skill в:

```text
.cursor/rules/
```

или подключите как глобальное правило проекта.

### Windsurf

Используйте Skill в качестве Workspace Rules.

### Другие LLM

Скопируйте содержимое `Skill.md` в системное сообщение модели.

## Принципы работы

Skill требует от модели:

* соблюдать научный стиль;
* не выдумывать источники;
* соблюдать структуру ВКР;
* формировать выводы по разделам;
* использовать академическую лексику;
* учитывать требования конкретного вуза.

## Ограничения

* Skill не заменяет методические указания кафедры.
* Skill не гарантирует прохождение проверки на заимствования.
* Пользователь обязан самостоятельно проверять актуальность источников и нормативных документов.
